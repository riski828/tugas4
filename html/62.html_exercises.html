Latihan HTML Dasar

Latihan 1: Heading & Paragraf

Buat judul halaman dengan <h1> lalu tambahkan paragraf menggunakan <p>.

<h1>Judul Halaman</h1>
<p>Ini adalah paragraf pertama saya.</p>

Latihan 2: Link

Buat link ke Google menggunakan <a>.

<a href="https://www.google.com">Buka Google</a>

Latihan 3: Gambar

Tampilkan gambar dengan tag <img> (gunakan link gambar dari internet).

<img src="https://via.placeholder.com/150" alt="Contoh Gambar">

Latihan 4: Daftar

Buat daftar belanjaan menggunakan <ul> dan <li>.

<ul>
  <li>Apel</li>
  <li>Pisang</li>
  <li>Jeruk</li>
</ul>

Latihan 5: Form Input

Buat form dengan input nama dan tombol submit.

<form>
  Nama: <input type="text" name="nama"><br>
  <button type="submit">Kirim</button>
</form>
